# Box it Up — Official Website

Box it Up connects consumers with surplus food from local restaurants, bakeries, 
and stores at discounted prices — reducing food waste while helping businesses 
recover costs. An Asian adaptation of the surplus-food marketplace model 
(à la Too Good To Go), starting with the Taiwan market.

This repository contains the official marketing and web platform for Box it Up.

## Features

- **Landing page** — product introduction and value proposition
- **Multi-language support** — English and Traditional Chinese
- **Waitlist signup** — early user registration, stored via Supabase
- **Responsive design** — optimized for mobile and desktop

## Tech stack

Next.js · Tailwind CSS · Supabase
